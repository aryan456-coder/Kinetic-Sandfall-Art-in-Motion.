# 🌌 Kinetic Sandfall – Art in Motion

An interactive generative art experience that simulates flowing sand-like particles, creating mesmerizing motion patterns directly in the browser.

## 🎨 About the Project

Kinetic Sandfall is a creative coding experiment that visualizes motion using particle-based animation. It mimics the behavior of falling sand and fluid motion, producing calming and dynamic visual effects.

Kinetic art focuses on movement—either real or simulated—to create engaging visual experiences.

---

## ✨ Features

- 🎮 Interactive particle movement
- 🌊 Smooth flowing “sandfall” animation
- 🖱️ Mouse-based interaction (if applicable)
- 🎨 Minimal and aesthetic UI
- ⚡ Real-time rendering in browser

---

## 🛠️ Tech Stack

- HTML5
- CSS3
- Vanilla JavaScript 

---

## ⚙️ How It Works

- JavaScript generates and updates particle positions
- Physics-like behavior simulates gravity and flow
- Browser renders frames continuously for smooth animation
